# 🚀 A Boa - Frontend

Aplicação web do projeto **A Boa**, uma plataforma para divulgação de eventos, ideias e "qual é a boa de hoje" 😄

---

## 📌 Sobre o projeto

O **A Boa** é uma aplicação frontend desenvolvida com **Vite + React + TypeScript + Tailwind CSS** para consumir uma API e exibir conteúdos dinâmicos como eventos, posts e interações dos usuários.

O objetivo é criar uma experiência leve, moderna e com foco em comunidade e descoberta de novidades.

---

## 🧠 Tecnologias utilizadas

* Vite ⚡
* React
* TypeScript
* Tailwind CSS
* HTML5
* Consumo de API REST

---

## ⚙️ Como rodar o projeto localmente

### 1. Clone o repositório

```bash
git clone git@github.com:cauacouto/a-boa-frontend.git
```

### 2. Entre na pasta

```bash
cd a-boa-frontend
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Rode o projeto em desenvolvimento

```bash
npm run dev
```

### 5. Build para produção

```bash
npm run build
```

### 6. Preview da build

```bash
npm run preview
```
